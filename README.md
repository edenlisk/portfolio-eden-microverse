1. Download [git](https://github.com/git-for-windows/git/releases/download/v2.38.1.windows.1/Git-2.38.1-64-bit.exe)

   <code>git@github.com:edenlisk/portfolio-eden-microverse.git</code> and you're ready to go!!!
